# Overall Project Description 
project for CSC3170 Database System
大三CUHKSZ数据库课程作业

CUHK(SZ) Dormitory Management System
使用的相关技术是java springboot + mysql + front-end static web pages。
该管理系统分为学生端和管理员端，主要实现了注册登录、查看宿舍分配情况、换宿申请、宿舍费用查询、维修申请五大功能。
登录校验使用了简单的http session技术，密码加密使用了Bcrypt，确保数据库中存储的以及实时登录比对的是加密后的内容。
